# Basic webpack configuration.
***
## How to run.
>$ yarn install
>$ yarn dev
***
## About the project.
**This is basic webpack configuration:**
* webpack;
* babel;
* mini-css-extract-plugin;
* CSS;
* SCSS;
* postcss (autoprefixer, css-mqpacker, cssnano);
* webpack-merge;
* copy-webpack-plugin;
* html-webpack-plugin;
* code Splitting, vendors.js;
* connecting and processing fonts.
